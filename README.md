# Link Generator - Location Tracker

## Project Overview
Link Generator - Location Tracker is a simple web-based application designed to generate unique tracking links. Users can create a link that, when accessed, will fetch their current geographical location and display it on a map. The application uses HTML, CSS (via Tailwind CSS), and JavaScript, and is user-friendly for both desktop and mobile platforms.

## Installation

To set up the project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd link-generator-location-tracker
   ```

2. **Open `index.html`:**
   Use your preferred web browser to open the `index.html` file to access the Link Generator interface.

## Usage

1. **Generate a Tracking Link:**
   - Click the "Generate Link" button on the main page to create a unique tracking link.
   - Copy the generated link using the "Copy Link" button.

2. **Trace Location:**
   - Open the generated link in a web browser.
   - Allow the application to access your location. 
   - Your latitude and longitude will be displayed on the page along with a Google Maps view of your current location.

## Features

- Generate unique links for tracking locations.
- Display geographical coordinates (latitude and longitude).
- Show the current location on a Google Map.
- User-friendly interface and design using Tailwind CSS.

## Dependencies

This project utilizes the following dependencies for its frontend development:

- **Tailwind CSS** for styling: [Tailwind CSS CDN](https://cdn.tailwindcss.com)
- **Font Awesome** for icons: [Font Awesome CDN](https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0-beta3/css/all.min.css)
